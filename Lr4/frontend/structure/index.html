Структура проекта (Frontend)

Основные файлы: - src/main.js — создание приложения, подключение Pinia, Router, Vuetify - src/App.vue — оболочка, AppBar, Drawer, router-view - src/router/index.js — маршруты, гард requiresAuth/requiresGuest
router/index.js - src/plugins/axios.js — конфиг Axios, интерсепторы
plugins/axios.js - src/plugins/vuetify.js — тема Vuetify
plugins/vuetify.js - src/services/api.js — обёртки над эндпоинтами
services/api.js - src/stores/auth.js — Pinia store для авторизации
stores/auth.js - src/views/* — страницы

Примеры использования API

Практические примеры работы с API системы управления типографией.

Настройка

Получение токена аутентификации

curl -X POST http://localhost:8000/api/auth/token/login/ \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "password123"
  }'

Ответ:

{
  "auth_token": "9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b"
}

Использование токена в запросах

TOKEN="9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b"

curl -X GET http://localhost:8000/api/books/ \
  -H "Authorization: Token $TOKEN"

Примеры на Python

Использование requests

import requests

BASE_URL = "http://localhost:8000/api"
TOKEN = "9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b"

headers = {
    "Authorization": f"Token {TOKEN}",
    "Content-Type": "application/json"
}

# Получение списка книг
response = requests.get(f"{BASE_URL}/books/", headers=headers)
books = response.json()
print(f"Всего книг: {books['count']}")

# Создание автора
author_data = {
    "first_name": "Лев",
    "last_name": "Толстой",
    "middle_name": "Николаевич",
    "birth_date": "1828-09-09"
}
response = requests.post(f"{BASE_URL}/authors/", headers=headers, json=author_data)
author = response.json()
print(f"Создан автор: {author['full_name']}")

# Получение финансовой сводки
response = requests.get(
    f"{BASE_URL}/financial/summary/",
    headers=headers,
    params={"start_date": "2023-01-01", "end_date": "2023-12-31"}
)
summary = response.json()
print(f"Баланс: {summary['balance']}")

Примеры на JavaScript (fetch)

const BASE_URL = 'http://localhost:8000/api';
const TOKEN = '9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b';

const headers = {
  'Authorization': `Token ${TOKEN}`,
  'Content-Type': 'application/json'
};

// Получение списка книг
fetch(`${BASE_URL}/books/`, { headers })
  .then(response => response.json())
  .then(data => {
    console.log(`Всего книг: ${data.count}`);
    data.results.forEach(book => {
      console.log(`- ${book.title}`);
    });
  });

// Создание финансовой записи
const financialData = {
  date: '2023-05-15',
  transaction_type: 'income',
  category: 'book_sales',
  amount: '250000.00',
  description: 'Продажа книг'
};

fetch(`${BASE_URL}/financial/`, {
  method: 'POST',
  headers,
  body: JSON.stringify(financialData)
})
  .then(response => response.json())
  .then(data => {
    console.log('Создана финансовая запись:', data);
  });

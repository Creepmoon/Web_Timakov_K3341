API Endpoints - Книги

Управление данными книг.

Базовый URL

/api/books/

Список книг

Получение списка всех книг.

Endpoint: GET /api/books/

Требуется: Аутентификация

Query параметры: - status - фильтр по статусу (draft, editing, design, printing, published, archived) - author - фильтр по ID автора - search - поиск по названию, ISBN или описанию

Пример запроса:

GET /api/books/?status=published&author=1

Создание книги

Создание новой книги.

Endpoint: POST /api/books/

Требуется: Аутентификация

Параметры запроса:

{
  "title": "Евгений Онегин",
  "isbn": "978-5-17-123456-7",
  "author_ids": [1, 2],
  "editor_id": 1,
  "pages": 300,
  "publication_date": "2023-05-15",
  "print_date": "2023-05-10",
  "print_run": 5000,
  "price": "500.00",
  "cost": "200.00",
  "status": "published",
  "description": "Роман в стихах..."
}

Доступные значения status: - draft - Черновик - editing - Редактирование - design - Дизайн - printing - Печать - published - Опубликована - archived - Архив

Примечание: - author_ids - массив ID авторов - editor_id - ID редактора (должен быть сотрудником с должностью "editor") - Все поля кроме обязательных могут быть пустыми

Обновление книги

Endpoint: PUT /api/books/{id}/ или PATCH /api/books/{id}/

Требуется: Аутентификация

Удаление книги

Endpoint: DELETE /api/books/{id}/

Требуется: Аутентификация

Ответ (204 No Content): Пустое тело ответа

Статистика по книгам

Получение статистической информации о книгах.

Endpoint: GET /api/books/statistics/

Требуется: Аутентификация

Ответ (200 OK):

{
  "total_books": 50,
  "by_status": [
    {
      "status": "published",
      "count": 30
    },
    {
      "status": "draft",
      "count": 10
    }
  ],
  "total_profit": 5000000.00,
  "average_price": 450.00
}

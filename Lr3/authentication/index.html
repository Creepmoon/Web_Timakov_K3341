Аутентификация

Система использует токен-аутентификацию через Djoser. Все API endpoints требуют аутентификации (кроме регистрации и входа).

Регистрация пользователя

Создание нового пользователя в системе.

Endpoint: POST /api/auth/users/

Параметры запроса:

{
  "username": "username",
  "email": "user@example.com",
  "password": "securepassword123",
  "re_password": "securepassword123"
}

Ответ (201 Created):

{
  "email": "user@example.com",
  "username": "username",
  "id": 1
}

Получение токена

Получение токена для аутентификации.

Endpoint: POST /api/auth/token/login/

Параметры запроса:

{
  "username": "username",
  "password": "securepassword123"
}

Или с email:

{
  "email": "user@example.com",
  "password": "securepassword123"
}

Ответ (200 OK):

{
  "auth_token": "9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b"
}

Использование токена

После получения токена, добавляйте его в заголовок всех запросов:

Authorization: Token 9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b

Выход из системы

Удаление токена (выход).

Endpoint: POST /api/auth/token/logout/

Требуется: Аутентификация

Ответ (204 No Content): Пустое тело ответа

Информация о текущем пользователе

Получение информации о текущем аутентифицированном пользователе.

Endpoint: GET /api/auth/users/me/

Требуется: Аутентификация

Ответ (200 OK):

{
  "id": 1,
  "username": "username",
  "email": "user@example.com",
  "first_name": "",
  "last_name": ""
}

Обновление информации о пользователе

Endpoint: PATCH /api/auth/users/me/

Требуется: Аутентификация

Параметры запроса:

{
  "first_name": "Иван",
  "last_name": "Иванов"
}

Изменение пароля

Endpoint: POST /api/auth/users/set_password/

Требуется: Аутентификация

Параметры запроса:

{
  "current_password": "oldpassword",
  "new_password": "newpassword123",
  "re_new_password": "newpassword123"
}
